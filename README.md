# Global Energy Transition & Sustainability Analysis

An end-to-end Data Analytics project focused on analyzing global energy consumption, renewable energy adoption, fossil fuel dependency, carbon intensity, and sustainability trends across countries using SQL and Power BI.

## Project Overview

This project analyzes how countries are transitioning toward renewable energy while balancing economic growth and environmental sustainability. The analysis was performed using SQL for data cleaning, transformation, and trend analysis, while Power BI was used for interactive dashboard creation and visualization.

## Key Business Problems Solved

* Which countries experienced the highest growth in energy consumption?
* Which countries are leading renewable energy adoption?
* Which countries reduced carbon intensity most effectively?
* Which economies achieved GDP growth while reducing emissions?
* Which countries remain highly dependent on fossil fuels?
* Which countries have the most diversified energy mix?

## Data Cleaning & Preparation

* Handled null and inconsistent records
* Validated percentage-based columns
* Filtered incomplete country-year records
* Performed data consistency and quality checks
* Structured data for time-series analysis and reporting

## SQL Analysis Performed

* Energy consumption trend analysis
* Renewable adoption growth analysis
* Carbon intensity reduction analysis
* Fossil fuel dependency analysis
* Consecutive renewable growth trend detection
* GDP vs sustainability comparison
* Energy diversification analysis using HHI Index
* Country-wise ranking and comparative trend analysis

## SQL Concepts Used

* Common Table Expressions (CTEs)
